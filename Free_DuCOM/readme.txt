(description for the sample program)
- This sample provides a minimal example that reads a sparse matrix in HB
  format and solves it using the DuCOM-COM3 sparse solver.
- If no RHS is provided, the code automatically generates one internally.

(note)
To prevent virus infection, the BuildKit.zip is password-protected.
Please contact info@sdesignlab.co.jp to get password.

(limitation)
This version uses a standard sequential solver, which means the computation
speed becomes very slow for larger matrices.

Note)
   The checking data file for Mindlin joint interface element subjected to 
UNI-AXILAL pull-out.  The attached plate element of elasticity
transfers axial force to the joint interface.  The force is applied uniformly
in the in-plane direction.
Note) 
   This data file is to check the model of steel model and the cyclic 
MK-Mishima's model.  The material index (=5) and data sequence is a little 
changed from standard model.         

Note) 
    This universal model represents pull-out from both sides of concrete.
Then, when you consider single sided pull-out like standard model, please
define a half of the real diameter as an equivalent one.
    This sample specifies 0.5(cm) as the bar size.  This is equivalent to 
1.0(cm) in the standard constitutive model.
